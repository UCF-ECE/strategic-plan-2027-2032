# ECE Strategic Plan 2027–2032

**Department of Electrical and Computer Engineering**  
University of Central Florida · College of Engineering and Computer Science

## Overview

This repository hosts the web-based materials for the ECE Strategic Plan 2027–2032, served via GitHub Pages.

## Structure

| File | Description |
|------|-------------|
| `index.html` | Landing page with links to both views |
| `slides.html` | Self-guided 15-slide overview for faculty |
| `plan.html` | Full interactive plan website *(add your existing artifact here)* |

## Setup

1. **Add your plan website**: Save your existing plan artifact as `plan.html` in the repo root.
2. **Enable GitHub Pages**: Go to **Settings → Pages → Source** and select `Deploy from a branch` → `main` → `/ (root)`.
3. **Access**: Your site will be available at `https://<org>.github.io/<repo>/`

## Feedback Survey

The slides link to a Qualtrics feedback survey with section-specific routing via URL parameters:

- `?section=overall` — Overall plan feedback
- `?section=p1` through `?section=p4` — Priority-specific feedback
- `?section=resources` — Resources feedback
